## Description

> This is a simple password protected file sharing website.

## Use

> make sure you have <strongNodeJs</strong> installed on your system.

```
  1. node v19.1.0 or greater 
  2. configure mongodb 
  3. fork repo 
  4. clone repo 
  5. npm install
  6. star repo to follow future changes 
```
